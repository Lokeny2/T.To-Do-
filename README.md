# Your Reliable Task Custodian | To-Do List App

## Overview

A simple to-do list application built with vanilla JavaScript. This project demonstrates DOM manipulation, event handling, and state management without any external libraries or frameworks.

## Live Demo

[View To-Do List App]https://lokeny2.github.io/T.To-Do-/

## Technologies Used

- HTML5
- CSS3 (hand-written, no frameworks)
- Vanilla JavaScript (no libraries)

## Features

- **Add Tasks** – Type a task and click "Add Task" or press Enter
- **Delete Tasks** – Remove individual tasks with the Delete button
- **Input Validation** – Prevents adding empty tasks
- **Empty State** – Shows a friendly message when no tasks exist
- **Keyboard Support** – Press Enter to add tasks quickly
- **Mobile Responsive** – Works on all screen sizes (≤ 400px stacks inputs vertically)
- **Focus Management** – Auto-focuses on input after adding a task

## What I Learned

- **DOM Manipulation** – Creating, adding, and removing elements dynamically
- **Event Listeners** – Handling clicks, keypresses, and keyboard events
- **State Management** – Tracking and updating the task list state
- **Conditional Logic** – Showing/hiding the empty message based on list length
- **CSS Styling** – Hand-crafted styles without Bootstrap
- **Mobile Responsiveness** – Media queries for small screens
- **Accessibility** – Proper form structure and keyboard support
- **Vanilla JS Patterns** – Functional programming approach
